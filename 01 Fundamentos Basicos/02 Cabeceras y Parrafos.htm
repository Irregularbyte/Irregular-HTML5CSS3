Otros elementos de frases

Elemento Ejemplo Uso
<abbr> abbr Se usa para marcar una abreviatura.
<b> Bold Texto en Negrita (Bold).
<cite> cita Otra forma de especificar una cita.
<code> codigo Una forma de citar codigo.
<dfn> definicion Se emplea para indicar la definicion de una palabra o termino.
<em> Enfatizar Se emplea para enfatizar un texto, normalmente en cursiva.
<i> cursiva Enfatiza un texto en cursiva.
<kbd> Texto en teclado Se utiliza para indicar un texto mecanizado en teclado, normalmente de fuente de tamaño fijo.
<mark> Ilumina un texto Se utiliza para iluminar/marcar un texto.
<samp> Sample text Muestra un texto de ejemplo, de fuente fija.
<small> Texto pequeño Reduce un texto.
<strong> Texto fuerte Marca un texto fuerte, normalmente en negrita.
<sub> Textosubindice Muestra el texto como subindice.
<sup> Textosuperindice Muestra el texto como superindice.
<var> Texto variable Muestra un texto variable, normalmente en cursiva.
